# FluentX

FluentX is a **modern C++ fluent-style UI framework** for building
**lightweight, native Windows applications** without heavy dependencies
like Qt or OpenGL.

> ⚠️ **Status:** Under active development (API is not stable yet)

---

## ✨ Goals

- Fluent, readable UI construction
- Native Win32-based rendering
- Minimal runtime overhead
- Debuggable, explicit ownership (no magic macros)
- Designed for engine-level control
